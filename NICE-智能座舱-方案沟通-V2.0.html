CHAPTER 01

项目背景与技术路径

NICE 智能座舱体验舱是华翔面向 2027 上海车展及后续全球车展打造的核心技术展示平台，并将持续迭代。我们以系统级总包方式，把核心零部件技术整合为可进入、可交互、可循环演示的 1:1 体验舱。

「健康舒适 + 智能表面」主题转译

技术实现路径：甲供核心技术输入 → 主题转译 → 4 大场景编排 → 软硬件系统集成 → 多模态交互 → 展示脚本与动画 → 体验舱制作 → 展会交付运维。在电子电器层面，建立以域控制器为核心、CAN FD 为主干的控制架构；在软件层面，开发多屏 HMI、场景引擎、健康数据可视化、语音交互、后台调试与一键复位；在保障层面，通过视频备用、后台触发、手动应急、备件与驻场确保首展稳定。

模型形式定义
形态

全舱 1:1，含完整外饰（前/后保/侧围/车顶/车门/尾门样式），可进入支持 4 人试乘。

座椅

前排 2 套（含 180° 旋转）+ 后排 2 套，合计 4 套零重力座椅。

边界

无实际尾门（仅样式）、车窗无升降、无底盘驱动系统；独立展台，可分拆运输，展馆安装 ≤8 小时。

CHAPTER 02

展示场景与体验脚本

每个场景构成「触发 → 系统响应 → 技术亮点 → 复位」完整闭环，覆盖 ≥2 种交互方式，并配套故事板、联动逻辑图、演示视频与验收用例。

🌟
SCENE 01 · P0
S1 — 迎宾场景
A 类 必达
触发：人体接近感应（毫米波/PIR）≤2m
  • ①P-HUD 播放带字标星河迎宾动画（15–20 秒）→ 主界面
  • ②整车氛围灯迎宾序列：由外向内渐亮，暖白→品牌色，≥8 分区流水
  • ③发光字标点亮 + 星空天幕激活（光纤）
  • ④语音播报「您好，欢迎乘坐 NICE 智能座舱」
备用演示：后台平板一键触发；迎宾动画视频备份；氛围灯可独立触发。
💚
SCENE 02 · P0
S2 — 健康舒适场景
A 类 必达
触发：主界面选「健康舒适」或语音「进入健康模式」
  • ①健康监测开始采集（甲供摄像头数据源，乙方域控取流）
  • ②健康画面显示于魔术屏/PAD/吸顶屏：红外画面 + 心率/HRV/血压/血氧
  • ③座椅调整至零重力（靠背角 + 腿托同步联动）
  • ④全车加热膜激活（扶手/门板/头顶/地毯，分区控温，上限 70℃）
  • ⑤氛围灯切换健康色 + 空调富氧离子/花粉过滤
责任边界：健康数据算法、准确性与医学边界由甲供方负责；我方负责取流驱动、数据解析、可视化与场景联动。现场数据不稳时支持模拟数据备用模式。
✨
SCENE 03 · P1
S3 — 智能表面场景
P1 重点
  • ①P-HUD/魔术屏/吸顶屏全部熄屏，体现「无屏即屏」消隐
  • ②星空顶关闭，扬声器下降收纳
  • ③IMSE 膜激活（隐藏触控 + 背光图案）
  • ④自发光缝线独立发光（独立控制，不与氛围灯联动）
  • ⑤多材质饰条（真木/真铝/碳纤维/金属拉丝）低亮度下呈质感
核心理念：以材料本身的光学与触觉特性展示华翔智能表面技术，与 S4 的机构运动形成对比。
🚀
SCENE 04 · P1
S4 — 智能空间场景
P1 重点
  • ①Console Body1（旋转出）+ Body2（滑出）展开
  • ②方向盘收缩（向前/向上折叠），模拟 L4 接管
  • ③前排座椅旋转 180° 面向后排（电控，≤15 秒，有提示音）
  • ④吸顶屏旋转至播放位 + 后排控制屏接管 + P-HUD 切自动驾驶信息
安全要求：座椅旋转有限位 + 防夹保护、旋转过程有提示音；方向盘收缩机械 + 电气双限位；一键复位全程 ≤30 秒。

CHAPTER 10

风险识别与应对

把高风险项前置识别、明确应对，是总包专业度的直接体现。我们在项目启动即建立风险台账并持续更新。

座椅 180° 旋转 — 安全与可靠性

应对：机械 + 电气双限位、旋转提示音；独立机构评审；≥100 次连续可靠性测试；现场工程师待命。

甲供件接口协议交付时点

应对：三级冻结（T0/T1/T2）+ 接口待确认清单周同步；接口未到位时优先推进不依赖项（造型/骨架/乙供硬件），协议到位即并行启动。

多屏多设备实时联动稳定性

应对：域控算力留余量；异步消息解耦；单场景压测后再联调；降级策略 + 全场景视频备用兜底。

健康数据真实性 vs 展示效果

应对：方案阶段明确 A/B 边界；实测稳定则展示真实值，否则书面确认采用模拟数据替代。

展会断电恢复（无 UPS）

应对：断电自锁 + 上电自动回零 + 参数记忆；如需「闪断不中断」可增配控制电 UPS（与甲方确认）。

车展展车级外观 — 多次运输保持

应对：定制防护包装；可调节安装方式；备件含易损外观件；安装手册提供缝隙调整方法。

需与华翔共同确认的开口项
① 取消 UPS 与技术要求 UPS 条款的偏差确认；② 加热膜各区面积/功率/控制器；③ 甲供专控（IMSE/星空天幕/健康）接口协议与接入方式；④ 魔术屏分辨率口径统一（9″ 规格）；⑤ 展馆供电容量（建议预留 16A×N 路）；⑥ 健康监测取流驱动 + 华翔 APP 的接口边界。
